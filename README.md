# iOS_UHF_Sample

iOS_UHF_Sample is a sample App to demonstrate how to use UHFSDK library.

## Supported Devices

* TS100
* TS100A
* TS800-00
* UR0250
* TS800-60

## Pre-requisites

* iOS 12.0+
* Xcode 12.0+
* Sample v3.1.5.2
* SDK v3.1.5.2

## Installation

* Please download this project and open with Xcode.

## Documents

* Please reference UHF SDK document [here](https://github.com/GIGA-TMS/doc_iOS_UHFSDK)
